# CS4210-Phishing-Website-Project
## Objective
This project uses ensemble models to predict the security of websites:
```
Phishing: -1 or Legitimate: 1
```
## Contents
1. Latex files: IEEE paper and diagrams regarding the outcomes of the project
2. Jupyter Notebook: ipynb file containing exploratory data analysis, modeling, and visualizations
3. Dataset: phishing-data.arff, from: [UCI Datasets](https://archive.ics.uci.edu/dataset/327/phishing+websites)
